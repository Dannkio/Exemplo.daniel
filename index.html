Questões dia 06/10

4.1- Quais os comandos para configurar usuário e email no git?

git config --global user.name "FIRST_NAME LAST_NAME" git config --global user.email e "MY_NAME@example.com"

4.2- Qual a diferença entre git e github?

Git é um software e github é uma plataforma baseada em web

4.3- O que faz o comando commit?

O comando git commit captura um instantâneo das mudanças preparadas do projeto no momento.

4.4- O que faz o comando push?

O comando push transfere commits do repositório local a um repositório remoto.

4.5- O que faz o comando pull?

Comando git pull é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.
